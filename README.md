# Next JS & Open AI / GPT: Next-generation Next JS & AI Apps

This repository contains the code and resources for the course **Next JS & Open AI / GPT: Next-generation Next JS & AI apps**. Below is a step-by-step walkthrough of the course structure and important notes about the repository.

## Course Walkthrough

This course covers the integration of Next.js with OpenAI's GPT to build next-generation AI-powered applications. The project evolves through several sections, each introducing new concepts and features.

### Important Notes About This Repository

1. **Branch Structure**:  
   - The branches in this repository represent different stages of the course, but they do not cover every detail of the course structure.  
   - The `master` branch has been merged with the final branch, `part-8.2`, which includes all the completed features up to that point.  

2. **Missing Sections**:  
   - The last 5 subsections of **Section 7** are not covered in this repository.  

3. **Package Manager**:  
   - This project uses `pnpm` as the package manager. Make sure you have it installed before setting up the project. 
   - To start the project run: `pnpm dev`
